# HumJ.TankRemake

决战坦克重制版（WIP）

# 介绍

这是一个对经典游戏《决战坦克》的非官方重制版。基于 Blazor Server，使用C#编写。支持单人模式，可以多人同时游玩。

# 运行环境

- .NET 8.0
- 最新版本的 Chrome、Firefox 或 Edge 浏览器

# 贡献

欢迎提交问题和建议，或者直接提交代码。请遵循以下规则：

- 提交代码前，请确保代码可以编译并通过测试。

# 已实现特性

- 音效系统

- 关卡地图
  - 读取原版地图文件

- 坦克
  - 坦克移动（地面、水面、冰面，速度等级）
  - 坦克底盘（水、冰）
  - 坦克与墙壁、地图边界碰撞检测
  - 坦克射击（主武器、副武器）

- 武器
  - 武器发射（随速度、火力等级变化）
  - 备弹数量

- 子弹
  - 移动
  - 地形破坏
  - 超时自爆

- 武器类型
  - 主武器（扫射改为随机散布）
  - 密集火力（扫射改为随机散布，保留概率不消耗备弹的特性）
  - 穿甲弹

# 已移除特性
